---------------------------------------------------------------------
------------------ Mise en place de l'application -------------------
---------------------------------------------------------------------

1 - Démarrage de l'API
   * Ouvrir un terminal
   * Ce placer dans le répertoire suivant : FlaskDev/sources
   * Lancer l'API avec la commande : python api.py

2 - Démarrage du serveur web
   * Ouvir un nouveau terminal ou un nouvel onglet
   * Ce placer dans le répertoire suivant : FlaskDev/static
   * Lancer le serveur avec la commande : python simpleHTTPserver.py

3 - Accès au site
   * Le service web est accessible sur le port 8080
   * L'API est accessible sur le port 5000
   * URL d'accès : http://127.0.0.1:8080/index.html
